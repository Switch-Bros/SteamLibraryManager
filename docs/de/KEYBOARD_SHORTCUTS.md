# ⌨️ Tastenkürzel

Alle verfügbaren Tastenkürzel im Steam Library Manager.

---

## Dateiverwaltung

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `Strg+S` | Sammlungen speichern | Speichert in Cloud Storage oder localconfig |
| `Strg+R` | Daten aktualisieren | Lädt Spieldaten aus Steam neu |
| `F5` | Daten aktualisieren | Alternative zu Strg+R |
| `Strg+Umschalt+S` | Backup erstellen | Exportiert ein Datenbank-Backup |
| `Strg+Q` | Anwendung beenden | |

## Navigation & Ansicht

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `Strg+F` | Suchleiste fokussieren | Springt zur Suche, sofort lostippen |
| `Strg+B` | Seitenleiste ein/aus | Blendet den Kategoriebaum ein oder aus |
| `Leertaste` | Detailbereich ein/aus | Zeigt/versteckt Spieldetails (wenn Baum keinen Fokus hat) |
| `Esc` | Leeren / Abwählen | **Gestuft:** 1. Druck leert die Suche, 2. Druck hebt die Auswahl auf |

## Spiel- & Kategorieverwaltung

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `Strg+A` | Alle Spiele auswählen | Wählt alle Spiele in der aktuellen Kategorie |
| `Entf` | Aus Kategorie entfernen | Entfernt ausgewählte Spiele aus der aktiven Kategorie |
| `F2` | Kategorie umbenennen | Benennt die aktuell gewählte Kategorie um |
| `Strg+I` | Bildbrowser | Öffnet den Artwork-Browser für das ausgewählte Spiel |

## Sammlungen & AutoCat

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `Strg+Umschalt+N` | Neue Smart Collection | Erstellt eine neue Smart Collection mit UND/ODER/NICHT-Regeln |
| `Strg+Umschalt+A` | Auto-Kategorisieren | Öffnet den AutoCat-Dialog mit 15+ Kategorisierungstypen |

## Werkzeuge

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `Strg+Umschalt+E` | Externe Spiele | Nicht-Steam-Spiele verwalten (Epic, GOG, Lutris, etc.) |
| `Strg+P` | Einstellungen | Anwendungseinstellungen öffnen |

## Hilfe

| Tastenkürzel | Aktion | Hinweise |
|--------------|--------|----------|
| `F1` | Benutzerhandbuch | Öffnet diese Dokumentation |

## Escape-Taste — Gestuftes Verhalten

Die `Esc`-Taste arbeitet in Stufen — mehrmaliges Drücken löst verschiedene Aktionen aus:

1. **Erster Druck:** Wenn die Suchleiste Text enthält → leert die Suche
2. **Zweiter Druck:** Wenn Spiele ausgewählt sind → hebt die Auswahl auf
3. **Dritter Druck:** Nichts (du bist bereits im Ausgangszustand)

So kannst du schnell zu einer sauberen Ansicht zurückkehren, ohne deinen Platz zu verlieren.
